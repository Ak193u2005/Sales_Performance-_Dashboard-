#  Sales Performance Dashboard (Power BI)

## Overview

This project is an interactive Sales Dashboard built using Power BI to analyze business performance across categories, regions, and time.

The focus was on creating a clean, interactive experience with meaningful insights using drillthrough, tooltips, and dynamic filtering.

---

## Key Features

* KPI Cards for:

  * Total Sales
  * Total Profit
  * Total Orders
  * Total Quantity

* Monthly Sales Trend (time-based analysis)

* Sales by Region (geographical performance)

* Sales by Category (distribution of sales)

* Profit vs Sales Analysis (understanding profitability vs volume)

* Drillthrough Page:

  * Category-level deep analysis
  * Product-level breakdown (Sales, Profit, Quantity)

* Custom Tooltips:

  * Region insights
  * Monthly breakdown
  * Category-level sub-analysis
  * Sales vs Profit via Category

* Interactive Slicers:

  * Region
  * Category
  * Year

---

## What I Focused On

* Building relationships between visuals
* Enabling deeper exploration using drillthrough
* Using tooltips to avoid clutter while adding insights
* Maintaining a clean and structured layout

---

## Tools & Technologies

* Power BI
* DAX (for calculated measures like Profit Margin, Avg Order Value)
* Visual Studio Code (used for layout planning and design reference)
